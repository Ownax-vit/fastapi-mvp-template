# FastAPI MVP Template

Производственный простой шаблон FastAPI с лучшими практиками, включающий примеры использования FastAPI, Pydantic и SQLAlchemy для небольших проектов

В каких случаях применять:
- MVP проекты и прототипирование
- Небольшие сервисы и микросервисы
- Внутренние инструменты и админки
- Backend для SPA приложений
- Учебные проекты
- Быстрый старт без долгой настройки
